# Privacy And Security Rules

- Child data is highest sensitivity.
- Server logs must not include child medical notes, incident details, raw registration payloads, message bodies, or payment secrets.
- No child/family/staff data may be sent to vendors or AI tools unless documented and approved.
- Secrets must never be committed.
- Public endpoints must be rate-limited or otherwise protected before production.
- Vendor usage must be documented in the product vendor register.
- Stop implementation if child-data, consent, retention, deletion, or breach impact is unclear.
